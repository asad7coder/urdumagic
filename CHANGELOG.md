# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-04-06

### Added

- Initial release of UrduMagic: Roman Urdu ↔ Urdu transliteration, LibreTranslate integration with caching and retries, script detection, DOM “magic” mode, RTL + Noto Nastaliq Urdu font loading, and a floating language switcher.
- TypeScript types, Vitest tests, plain HTML and React demos, and npm/CDN builds (ESM, UMD, IIFE).
