# Introduction

Welcome to **UrduMagic**, the lightweight solution for adding seamless Urdu language support to any web project.

## What is UrduMagic?

UrduMagic is a high-performance TypeScript library (~222KB gzipped) designed to solve a common problem: making the web more accessible to Urdu speakers. Whether you want to provide transliteration between Roman Urdu and Urdu script or offer full English-to-Urdu translation using its **10,000+ entry offline dictionary**, UrduMagic makes it easy.

## Why UrduMagic?

Urdu is one of the most widely spoken languages in the world, yet it remains underrepresented in modern web development. Most sites either ignore Urdu script or struggle with the complexities of RTL (Right-to-Left) layouts and script detection.

UrduMagic was built to:
- **Offline First:** Massive **10,000+ entry dictionary** for instant $O(1)$ lookups without internet.
- **Optimized Pipeline:** High-speed translation flow (Dictionary → Cache → DOM Mutation).
- **Hardened Security:** Built-in XSS protection and prototype pollution prevention.
- **Enable Magic Mode:** Automatically scan and switch visible text on your page with zero configuration.

## Key Features

- ✅ **Offline First:** Transliteration logic runs entirely in the browser.
- ✅ **RTL by Design:** Built-in support for Right-to-Left script layouts.
- ✅ **Smart Detection:** Automatically detect if text is English, Urdu, or Roman Urdu.
- ✅ **Dictionary Driven:** Powered by a local 10,000+ entry English-Urdu mapping.
- ✅ **MIT Licensed:** Free and open source for everyone.
