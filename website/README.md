# UrduMagic Documentation Portal 🪄

This is the official documentation and marketing portal for [UrduMagic](https://github.com/asad7coder/urdumagic).

## 🚀 Features
- **Next.js 15+** App Router architecture.
- **MDX Integration** for technical documentation.
- **UrduMagic Live Demo**: Real-time 3-way transliteration engine testing.
- **Full Offline Engine**: Zero API dependencies for translations.
- **Responsive RTL Design**: Native support for Urdu script layouts.

## 🛠️ Development

This project uses `urdumagic` as a local dependency (`file:..`).

1.  **Build the Library**: Run `npm run build` in the root directory.
2.  **Install**: Run `npm install` in this folder.
3.  **Run Dev**: `npm run dev`
