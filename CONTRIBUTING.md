# Contributing

Thanks for helping improve UrduMagic. This project targets Pakistani developers and learners who want simple English, Urdu, and Roman Urdu support in the browser.

## How to contribute

1. Fork the repository and create a branch from `main`.
2. Install dependencies with `npm install`.
3. Run tests with `npm run test:run` and build with `npm run build`.
4. Keep changes focused: one feature or fix per pull request when possible.
5. Match existing TypeScript style (`strict` mode, no `any`).
6. Add or update tests when you change transliteration rules, caching, translation behavior, or DOM magic mode.

## Dictionary and transliteration

The Roman → Urdu dictionary is hand-authored for this package. Do not copy word lists or rules from other open-source transliterators; extend our data with original research and common Pakistani spellings.

## Questions

Open a discussion or issue on GitHub if you are unsure whether a change fits the project goals.
